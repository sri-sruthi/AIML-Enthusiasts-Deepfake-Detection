# Deepfake Detection System – AIML Enthusiasts

## Setup Instructions

1. Clone or download this repository.  
   ```bash
   git clone https://github.com/<your-username>/AIML-Enthusiasts-Deepfake-Detection.git
   cd AIML-Enthusiasts-Deepfake-Detection
   
2. Install the required dependencies.

pip install -r requirements.txt

3. Ensure the dataset structure follows:

/DATASET/
  ├── train/
  ├── test/
  ├── training.json


4. Place the dataset inside your Google Drive and update the paths inside the notebook if required.
